## Getting started
1. Install Golang
2. Install just
3. Configure Git LFS

## Run the project
To run with both Templ and Tailwind hot-reloading:
```sh
just
```

> equal to `just dev`

To generate Templ files:
```sh
just templ
```

To generate Tailwind output:
```sh
just tailwind
```
